本文档介绍如何在惠州仲恺中学官网中添加和编辑文章内容。

## ✍️ 如何添加新文章

### 步骤 1：在 data.ts 中添加基础信息

打开 `src/lib/data.ts`，在对应的数组中添加基础信息：

```typescript
// 例如：添加新闻
export const NEWS_DATA: NewsItem[] = [
  {
    id: 7,  // 新的 ID
    title: "文章标题",
    date: "2024-01-15",
    category: "教务动态",
    summary: "文章摘要...",
    image: "https://example.com/image.jpg"
  },
  // ...
];
```

### 步骤 2：创建 Markdown 文件

在对应的目录下创建 `{id}.md` 文件：

```
content/news/7.md
```

## 📝 Markdown 文件格式

### 基本结构

```markdown
---
title: 文章标题
date: 2024-01-15
author: 作者名称
tags: [标签1, 标签2]
---

# 正文标题

正文内容...
```

### Frontmatter（文件头部元数据）

| 字段 | 说明 | 必填 |
|------|------|------|
| `title` | 文章标题 | 否（默认使用 data.ts 中的标题） |
| `date` | 发布日期 | 否 |
| `author` | 作者 | 否 |
| `tags` | 标签数组 | 否 |

## 📖 Markdown 语法指南

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 段落

```markdown
这是第一段。

这是第二段。
```

### 粗体和斜体

```markdown
**粗体文字**
*斜体文字*
***粗斜体***
```

### 链接

```markdown
[链接文字](https://example.com)
```

### 图片

```markdown
![图片描述](https://example.com/image.jpg)
```

### 列表

无序列表：
```markdown
- 项目一
- 项目二
- 项目三
```

有序列表：
```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 引用

```markdown
> 这是一段引用文字
```

### 代码

行内代码：
```markdown
这是 `代码` 示例
```

代码块：
```markdown
```javascript
console.log("Hello World");
```
```

### 表格

```markdown
| 标题1 | 标题2 |
|-------|-------|
| 内容1 | 内容2 |
| 内容3 | 内容4 |
```

### 水平线

```markdown
---
```

## ⚠️ 注意事项

1. **文件名必须与 ID 一致**：如果 data.ts 中 id 是 7，文件名必须是 `7.md`

2. **编码格式**：文件必须使用 UTF-8 编码

3. **图片建议**：
   - 使用 CDN 托管图片
   - 图片 URL 使用 HTTPS
   - 建议图片宽度不超过 1200px

4. **SEO 优化**：
   - 标题简洁明了
   - 摘要控制在 150 字以内
   - 合理使用标签

5. **性能考虑**：
   - 单篇文章建议不超过 5000 字
   - 图片数量建议不超过 20 张
   - 避免过大的表格
